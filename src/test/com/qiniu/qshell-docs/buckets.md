# 简介

`buckets` 指令可以获取当前账号下所有的空间名称并输出。

# 格式

```
qshell buckets
```

# 鉴权

需要在使用了`account`设置了`AccessKey`和`SecretKey`的情况下使用。

# 参数

无参数。

# 示例

```
$ qshell buckets
```

输出：

```
gitbook
if-pbl
if-pri
jemydemob
qbug
qdemo
qdemo2
qdisk
qdoc
qiniu-lab
qiniudevspace
qiniulab-private
qiniulab-public
qlog
qphoto
qprivate
```
